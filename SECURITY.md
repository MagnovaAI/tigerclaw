# Security policy

tigerclaw is pre-alpha. There is no supported release yet.

## Reporting

Report suspected vulnerabilities privately by emailing the maintainer listed in `Cargo`-equivalent metadata (`build.zig.zon`) or by opening a GitHub security advisory on the repository.

Please do not open public issues for vulnerabilities before a fix is available.

## Scope

Once the project has a tagged release, this file will list supported versions, a disclosure timeline, and the coordinated-disclosure process. Until then, best-effort triage applies.
