# Contributing

Early days. PRs welcome but expect churn.

## Dev

```sh
zig build test
zig build run
```

## Conventions

- Zig `0.16.0` stable
- Conventional commits (`feat:`, `fix:`, `chore:`, `refactor:`, `docs:`)
- Small, focused PRs
